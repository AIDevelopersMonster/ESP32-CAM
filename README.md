# ESP32-CAM

Учебный проект по ESP32-CAM: установка Arduino IDE, настройка ESP32 Core, подключение модуля, загрузка скетчей, Wi-Fi AP/STA, диагностика и материалы для уроков/видео.

## Структура

- `00_Карта_проекта_README` — карта проекта и навигация.
- `01_Arduino_IDE_и_ESP32_Core` — установка Arduino IDE, JSON ESP32 Core, выбор платы.
- `02_Платы_модули_и_подключение` — наши модули, USB-TTL, питание, RST, GPIO0.
- `03_Скетчи_Arduino_и_код` — CameraWebServer, AP Mode, STA Mode, эксперименты.
- `04_Загрузка_прошивка_и_драйверы` — драйверы, режим загрузки, особенности RST.
- `05_WiFi_AP_STA_IP_и_браузер` — SSID/PASSWORD, Serial Monitor, запуск в браузере.
- `06_Схемы_пины_таблицы` — схемы, распиновки и таблицы.
- `07_Фото_железа_и_монтаж` — фото железа и монтаж.
- `08_Видео_демо_и_скриншоты` — видео, демо и скриншоты.
- `09_Ошибки_диагностика_Troubleshooting` — компиляция, загрузка, камера, Wi-Fi.
- `10_Документация_datasheets_ссылки` — datasheets и полезные ссылки.
- `11_Учебные_материалы_и_методичка` — инструкция, презентация, сценарий урока, публикации.
- `12_Архив_старые_версии` — архив старых версий.
