# ESP32-CAM

Учебный проект по ESP32-CAM: установка Arduino IDE, настройка ESP32 Core, подключение модуля, загрузка скетчей, Wi-Fi AP/STA, диагностика и материалы для уроков/видео.

Структура проекта будет разложена по тематическим папкам: от установки среды до troubleshooting, схем, фото, документации и учебных материалов.
