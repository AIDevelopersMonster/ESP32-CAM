# ESP32-CAM: WiFiManager GUI, IP и кнопка запуска браузера

Этот вариант заменяет ручное редактирование `ssid` и `password` в скетче на веб-настройку Wi-Fi через библиотеку **WiFiManager**.

## Что получается

1. При первом запуске ESP32-CAM создает временную точку доступа `ESP32-CAM-SETUP`.
2. Пользователь подключается к этой точке с телефона или компьютера.
3. Открывается страница настройки Wi-Fi.
4. Пользователь выбирает домашний роутер и вводит пароль.
5. ESP32-CAM подключается к роутеру.
6. В Serial Monitor выводится IP адрес.
7. По адресу `http://IP_КАМЕРЫ/` открывается простая страница с кнопкой **Открыть видеопоток**.
8. Видеопоток доступен по адресу `http://IP_КАМЕРЫ/stream`.

## Важно

ESP32-CAM не может сама запустить браузер на компьютере. Микроконтроллер не управляет программами на ПК. Поэтому правильная схема такая:

```text
ESP32-CAM получает IP -> показывает веб-страницу -> пользователь нажимает кнопку -> браузер открывает поток
```

## Что установить в Arduino IDE

Открой:

```text
Sketch -> Include Library -> Manage Libraries
```

Найди и установи:

```text
WiFiManager by tzapu
```

Также должен быть установлен ESP32 Core для Arduino.

## Где скетч

Скетч лежит здесь:

```text
03_Скетчи_Arduino_и_код/04_Эксперименты_и_версии/ESP32_CAM_WiFiManager_Stream/ESP32_CAM_WiFiManager_Stream.ino
```

## Как пользоваться

1. Открой скетч в Arduino IDE.
2. Выбери плату:

```text
AI Thinker ESP32-CAM
```

3. Загрузи скетч в плату.
4. Отключи `GPIO0` от `GND` после прошивки.
5. Перезагрузи ESP32-CAM.
6. Открой Serial Monitor на скорости:

```text
115200
```

7. Если Wi-Fi еще не настроен, подключись к сети:

```text
ESP32-CAM-SETUP
```

Пароль точки настройки:

```text
12345678
```

8. В браузере открой:

```text
http://192.168.4.1
```

9. Выбери свой роутер, введи пароль и сохрани.
10. После подключения в Serial Monitor появится IP, например:

```text
Camera page: http://192.168.1.55/
Camera stream: http://192.168.1.55/stream
```

## Как сбросить сохраненный Wi-Fi

Временно раскомментируй строку в скетче:

```cpp
// wm.resetSettings();
```

Чтобы получилось:

```cpp
wm.resetSettings();
```

Загрузи скетч один раз, затем снова закомментируй эту строку и загрузи обычную версию.
